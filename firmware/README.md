# elbow-exo


Using PlatformIO for low-level development of the elbow exoskeleton firmware.

To get started, install [PlatformIO IDE](https://platformio.org/install/ide?install=vscode) for Visual Studio Code.

Then, clone this repository and open it in VS Code. PlatformIO should automatically detect the project and set up the environment for you. If not, you can manually open the PlatformIO Home and navigate to the project.
